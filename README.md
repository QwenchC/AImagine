# AImagine

Android app project `AImagine`.

## Overview

AImagine is an Android application (Kotlin + Jetpack) located in this repository. It contains an Android Studio project with Gradle build files.

## Build & Run

Prerequisites:
- Java JDK (11+)
- Android SDK (installed via Android Studio)

To build from command line:

```bash
./gradlew assembleDebug
```

Open the project in Android Studio to run on an emulator or device.
